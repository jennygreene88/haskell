# Demo Details

## Fizz Buzz

### How to Run

#### Using GHCi

Open the GHCi command line.

Navigate to the fizzbuzz directory.

Load the fizzbuzz.hs module using this command:

'''
:l fizzbuzz
'''

Fizz Buzz using list comprehension:

'''
fizzbuzz
'''

Fizz Buzz using recursion:

'''
fizzbuzz_r
'''
